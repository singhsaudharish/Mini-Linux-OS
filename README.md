# Mini-Linux-OS
Mini OS Login System is a Java Swing-based desktop application that provides user authentication and launches a terminal-like environment after successful login.

A simple desktop-based **Mini Operating System Login Interface** built using **Java Swing**. This project provides user authentication, registration, and access to a terminal-like environment after successful login.

## 📌 Features

- 🔐 User Login Authentication
- 👤 User Registration
- 🖼️ Custom Application Icon
- 💬 Error Handling with Dialog Messages
- 🖥️ Java Swing GUI Interface
- 🚀 Opens Mini Linux Terminal after successful login

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT
- Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code)

### Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/mini-os-login.git
```

2. Navigate to the project folder:

```bash
cd mini-os-login
```

3. Compile the project:

```bash
javac *.java
```

4. Run the application:

```bash
java LoginScreen
```

## 🔑 Login Process

1. Enter your username and password.
2. Click **Login**.
3. If credentials are correct, the Mini Linux Terminal opens.
4. If credentials are invalid, an error message is displayed.

## 👤 Registration Process

1. Enter a new username and password.
2. Click **Register**.
3. User information is stored using the `UserManager` class.
4. A success message is displayed.
